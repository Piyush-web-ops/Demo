# Piyush Rai - Portfolio Website

This is my personal portfolio website built using HTML, CSS, JavaScript, Docker, and Nginx.

## Skills
- AWS
- Docker
- Kubernetes
- Terraform
- Linux
- Jenkins
- DevOps

## Run Locally
git clone https://github.com/Piyush-web-ops/Demo.git
cd Demo
docker build -t portfolio .
docker run -d -p 80:80 portfolio
